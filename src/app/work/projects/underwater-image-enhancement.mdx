## Overview

Developed an advanced underwater image enhancement system designed to restore clarity and visibility in underwater photography by addressing the inherent challenges of underwater imaging such as color distortion, reduced contrast, and information loss.

## Problem Statement

Underwater images suffer from significant quality degradation due to:
- **Light absorption**: Water absorbs red wavelengths, causing color cast
- **Reduced contrast**: Limited visibility due to scattering particles
- **Information loss**: Critical details obscured by environmental factors
- **Color distortion**: Unnatural appearance affecting visual analysis

Traditional enhancement methods often amplify noise or lose important visual information.

## Solution & Implementation

### Key Features

- **Two-Stage Preprocessing**:
  - White balancing to correct color cast and restore true color representation
  - Contrast enhancement to restore visibility and detail definition

- **Intelligent Weight Mapping**:
  - Calculates weight maps based on luminance (brightness distribution)
  - Considers chromaticity (color information)
  - Incorporates saliency (importance of image regions)

- **Normalized Weight Fusion**:
  - Combines multiple weight maps to generate balanced normalized weights
  - Preserves important features while reducing artifacts

- **Multi-Scale Pyramid Processing**:
  - Constructs Gaussian pyramids for hierarchical analysis
  - Enables seamless blending across different image scales
  - Reduces boundary artifacts in fused results

### Technical Architecture

```
Input Image
    ↓
[Preprocessing Stage]
├─ White Balancing
└─ Contrast Enhancement

```

| Challenge | Solution |

| Computational efficiency | Optimized pyramid construction and blending algorithms for real-time processing |
